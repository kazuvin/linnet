# Linnet - ギターコード進行リファレンスアプリ

## プロジェクト概要

キーを設定し、ダイアトニックコードやモーダルインターチェンジしたコードをドラッグ&ドロップでコード進行として配置。選択したコードに応じてギター指板上で構成音・使えるスケールを視覚的に確認できるリファレンスアプリ。

### コアコンセプト

- **リファレンスツール**: DAW のような作曲機能は持たない。コード理論の学習・確認・アイデア出しのための道具
- **幅広いユーザー層**: 初心者の学習から上級者の素早い確認まで対応
- **レスポンシブ**: デスクトップ・モバイル両対応

### 主要機能（段階的に実装）

1. キー選択 → ダイアトニックコード・モーダルインターチェンジコードの一覧表示
2. コードのドラッグ&ドロップによるコード進行の組み立て
3. ギター指板上でのコード構成音・スケール表示

# 開発ルール(重要)

- 日本語を使うこと
- 複雑なタスクは分割してサブエージェントに委譲すること
- Task の最後には `/ci` を実行して CI を通すこと
- パッケージマネージャーは必ず `pnpm` を使うこと
- Skills は積極的に使うこと
- 不明点は必ず確認すること
- 後方互換性は保たなくて良い
